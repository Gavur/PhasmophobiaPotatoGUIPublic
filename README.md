Build instructions:

Run RUNME.bat as ADMINISTRATOR and enter the FULL path to your MelonLoader folder. e.g. <br />
``A:\SteamLibrary\steamapps\common\Phasmophobia\MelonLoader``

Open ``PhasmophobiaPotatoGUI.sln`` and build.
